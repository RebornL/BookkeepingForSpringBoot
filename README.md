# BookkeepingForSpringBoot



## 前言

此项目为BookkeepingBackend项目的SpringBoot简单实现，主要采用SpringBoot2+mybatis重新原有的REST接口，传输数据格式均是json数据格式。



SpringBoot2确实很不错，很简洁和很容易上手，而且Mybatis很容易嵌入进去。Mybatis基本采用注解的配置方式，比xml配置方式简介很多。
